How do I get this thing to work?

This should take 5 minutes to setup by yourself. It might be a little complicated to get setup, but follow the steps below and you'll be set. Part 1: Getting Started

Just a few steps to get it setup:


  1. Click the Download button on this page that gets you a ZIP file. Please download the ZIP to somewhere on your desktop, or somewhere easy to find.

  2. Unpack the ZIP, by clicking on it.

  3. You're going to see a folder full of files with code that makes the tool work.

  4. Click only on the file that's called YeezyCopper.JS.

  5. Now the file should be open in an editor or something like this:

  6. Edit the information to reflect your shipping & CC details. It should be easy-- I put hints to the right of each field.

  7. SAVE the file once you typed everything in. Also, make sure that you didn't have any typos.

Part 2: Finishing it Up

You're almost done. Now we have to put the tool into Chrome and run it on Adidas.com:


  1. Go to chrome://extensions/.

  2. Click the Developer Mode checkbox.

  3. Click "Load Unpacked Extension" and then just select the folder with all the files you downloaded.

  4. Click Install.

  5. You have it setup. Good job!!


Part 3: Try it out!

  1. You need to be already logged into Adidas.com

  2. Use the extension on any page that you want to buy shoes from. This extension is designed for Yeezys, but it works the same on any other pair of shoes you want from Adidas.

  3. Sometimes you're going to need to fill out something called a CAPTCHA, so after that happens, press the G key on your keyboard. Then, the software will fill out the forms with all the info you typed in earlier and cop your shoes.

  4. Good luck!

Notes/Troubleshooting:

Did something break? Check below for some help:


  • If the extension is disabled or is messing up for whatever reason, delete it in chrome://extensions and redo Step 2.
  • Make sure you have NO typos. It won't work if you have a typo.
  • Make sure you are logged into Adidas.com before using this.
